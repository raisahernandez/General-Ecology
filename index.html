General Ecology

The following activites were designed to introduce data-centered approaches using R programming language to General Ecology students.

Super-short introductory activities to R. Each activity builds into the previous one:

Intro I: Importing data to RStudio and getting ready to analize it

Intro II: Data management using tidyverse

Intro III: Data visualization using ggplot2


Quantitative activities for General Ecology:

Solar radiation and global temperatures

Biomes and climate diagrams

Adaptation to variable environments

Evolution and Adaptation

BIOMAAP Adaptation
